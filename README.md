# SKILLFORGE

**SkillForge** is a role-based Learning Management System (LMS) that leverages AI to enhance teaching efficiency and deliver a personalized, data-driven learning experience.

---

## 🔑 Key Features

### 👤 Role-Based Access
- **Admin**: Manage users, course catalog, feedback, and announcements  
- **Instructor**: Create/manage courses, generate AI-based quizzes, and analyze student performance  
- **Student**: Track progress, receive AI recommendations, and view performance trends  

---

## 🧠 Intelligent Capabilities
- **AI Quiz Generator** – Automatically generates quizzes from course topics  
- **Analytics Engine** – Provides insights like learning trends and top performers  
- **Targeted Announcements** – Role-based communication system  
- **Feedback System** – Integrated feedback loop for continuous improvement  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- React Router DOM  
- Axios  
- Vanilla CSS (responsive UI with modern design)

### Backend
- Spring Boot (Java 17+)  
- Spring Data JPA (Hibernate)  
- RESTful APIs  

### Database
- MySQL  

### Tools & Security
- Maven, npm  
- JWT Authentication  
- JUnit, React Testing Library  

---

## ⚡ Highlights
- Scalable and modular architecture  
- Secure role-based access control  
- AI-powered learning and analytics  
- Personalized learning experience

## ▶️ Getting Started (Basic)

### Backend
```bash
cd backend
mvn spring-boot:run
```
### Frontend
```bash
cd frontend
npm install
npm start
```
